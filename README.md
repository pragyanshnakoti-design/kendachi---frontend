# KENDACHI

Kendachi is now organized into simple folders so it is easier to understand:

- `backend/` = Node.js + Express backend
- `frontend/` = React + Vite frontend
- `frontend/src/styles/` = CSS files

## Local Run

1. Install packages:
   `npm install`
2. Start backend + frontend:
   `npm run dev`
3. Open:
   `http://localhost:5173`

Demo admin:

- Email: `admin@company.org`
- OTP appears on the login screen in local dev mode
